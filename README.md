## 🎨 How to Customize

### Update Affiliate Links in Deals Hub
1. Open `index.html`
2. Find: `https://your-affiliate-link-X.com`
3. Replace with your actual affiliate links

### Change Colors
Edit CSS colors like `#667eea` (purple) or `#ff6a00` (orange) to match your brand.

### Add More Products
Duplicate product cards in `index.html` and update:
```html
<div class="card">
  <img src="product-image.jpg">
  <h3>Product Name</h3>
  <p>Product description</p>
  <a href="YOUR-AFFILIATE-LINK">Buy Now</a>
</div>
```

### Add More Time Zones to Clock
Edit the `timeZones` array in `digital-clock.html`:
```javascript
{ name: 'Your City', timezone: 'Timezone/City', emoji: '🌍' }
```
